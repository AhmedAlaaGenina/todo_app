# To-Do App
## _Flutter App to add tasks and mange it_

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

To-Do App is Aflutter App That help to mange tasks 

- Flutter
- SQflit
- ✨BLOC✨

## Features

- Add tasks
- Done Tasks
- Archived Tasks
- Delete Tasks
